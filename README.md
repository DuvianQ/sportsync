# Sportsync - Prototipo

Prototipo funcional para la entrega del proyecto. HTML/CSS/JS simple.

## Cómo ejecutar localmente
1. Abrir `index.html` con Live Server (o abrir en navegador).
2. Para desplegar: subir a GitHub y activar GitHub Pages (branch `main`, root).

## Funcionalidades implementadas
- Login demo (admin / demo)
- Dashboard con métricas y gráfico simulado
- Inventario con CRUD básico (añadir / editar / eliminar)
- Registro de ventas y devoluciones (afecta stock)
- Gestión simple de proveedores
